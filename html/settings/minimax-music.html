📖 歌词结构教程：14 种段落标签的含义与使用时机

在歌词最前面用 [标签] 单独占一行，告诉模型"接下来这一段是什么类型"。 模型会据此调整旋律走向、配器密度、能量曲线。 标签必须用英文方括号写英文名，中文按钮只是显示用，插入后真正写入的还是英文。

🏗 一首歌的常见骨架

最经典的流行结构（你 90% 的歌都可以套这个）：

[Intro]  →  [Verse]  →  [Pre Chorus]  →  [Chorus]
        →  [Verse]  →  [Pre Chorus]  →  [Chorus]
        →  [Bridge]  →  [Chorus]  →  [Outro]

电子/舞曲常见：[Intro] → [Build Up] → [Drop] → [Breakdown] → [Build Up] → [Drop] → [Outro]。

🎤 含人声的段落（写歌词的地方）
  • 🎤 主歌 [Verse]：叙事段，铺陈故事 / 情绪。每段 4–8 句，旋律相对平稳，能量中等。
    用法：每首歌通常 2–3 段 Verse，第 2 段 Verse 歌词不同但旋律可重复第 1 段。
  • 🎶 预副歌 [Pre Chorus]：副歌前的"引线"，能量从 Verse 慢慢推上去。1–2 句即可。
    用法：可选段落；用了能让副歌爆发更有冲击力，去掉也不影响结构。
  • 🎙️ 副歌 [Chorus]：全曲记忆点 / 情绪高峰，反复出现。一般写最朗朗上口的几句。
    用法：一首歌至少 2 次，推荐 3 次（第 3 次可升 key 或加和声）。
  • 🪝 Hook：钩子段，比 Chorus 更短更洗脑，常见于嘻哈/流行。可以代替 Chorus 或穿插出现。
    用法：1–2 句简短记忆点，比如"baby baby ohh"那种，专门用来洗脑。
  • 🌉 过门 [Bridge]：第二次副歌之后的"转折段"，情绪 / 调性变化，避免听觉疲劳。
    用法：通常 4–8 句，歌词内容与 Verse / Chorus 都不同；之后回到最后一次副歌。
🎼 纯演奏 / 过渡段（不写歌词或留空）
  • 🎼 前奏 [Intro]：开场氛围铺垫，0–15 秒。
    用法：基本每首都加，给听众一个进入歌曲的过渡。下面通常空一行不写歌词。
  • 🏁 尾奏 [Outro]：结尾收束，淡出 / 重复关键句 / 器乐 fade out。
    用法：基本每首都加，可以让模型自然结束而不是被切断。
  • 🎵 间奏 [Interlude]：段落之间的纯器乐过渡，比 Bridge 短，不带歌词。
    用法：Verse 1 → Verse 2 之间需要喘息时插入。
  • 🎻 器乐段 [Instrumental]：完全无人声的演奏段，常用于民谣 / 摇滚的"假尾奏"。
    用法：跟 Interlude 类似但更长更主动，可以理解为"把人声让位给乐器秀一段"。
  • 🎸 Solo：器乐独奏段（吉他 solo / 钢琴 solo / 萨克斯 solo 等）。
    用法：摇滚 / 爵士 / 蓝调风格里，常放在第二次副歌之后、Bridge 之前。
⚡ 电子 / 舞曲专用
  • 📈 递进 [Build Up]：能量从低到高的爬升段（白噪声 / 鼓点逐渐密集 / 滤波打开）。
    用法：Drop 之前必备，制造"快要爆炸"的预期感。
  • 💥 Drop：能量瞬间释放的高潮段，电子舞曲的核心爽点。
    用法：跟在 Build Up 之后；House / Dubstep / Trap 必备段落。
  • 🔻 Breakdown：把 Drop 之后的密集编排"拆解掉"，只剩极少元素（人声 / 键盘垫底）。
    用法：电子曲中常做"Drop → Breakdown → Build Up → Drop"循环，制造起伏。
  • ⏸ 间歇 [Break]：完全或几乎静音的短暂停顿（半拍 ~ 2 小节）。
    用法：在 Drop 之前一拍 [Break] 能极大增强落点的冲击力。

💡 实用 Tips
  • 标签 独占一行，前后各空一行可读性更好。
  • 纯器乐段（Intro / Outro / Interlude / Instrumental / Solo / Drop / Build Up / Breakdown / Break）下方不要写歌词，留空让模型自由发挥。
  • 同一标签可重复使用，模型会理解为"再来一段同类型的"。
  • 歌词字数控制在 1~3500 字符；段落越多，每段越短，节奏感越强。
  • 风格描述（上方 prompt）决定"曲风"，段落标签决定"结构"，两者配合才能稳定出货。
